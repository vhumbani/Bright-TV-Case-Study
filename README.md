# Bright-TV-Analysis
This repository contains detailed Tv analysis from 16th January 2016 to 01 April 2016. The goal is to undertand usage trends by age,race,province,daytime and months.

# Summary of Finding  Highest Viewership by Age from highest to lowest.
* Youth
* Adults
* Matured Adults
* Kids
* Retired
  
# Summary of Viewership 
* Race----Black is the top race in viewership.
* Gender---Males viewed tv more than females.
* Province---Gauteng has the highest viewership.
* Months---March has the highest viewership
* Daytime---Friday has the highest views.
